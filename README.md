# My Todo

---

MyToDo adalah sistem pencatatan aktivitas-aktivitas harian yang dapat berupa kegiatan, pengingat, dll.

### Tech Stack

- App Framework: Echo Golang
- Database: MariaDB
- ORM: GORM
- Authentication: JWT (Users)
- Code Structure: MVC
- Deployment: Google Cloud Platform (GCP)
